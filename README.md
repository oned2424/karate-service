# 空手動画学習プラットフォーム

CC-BYライセンスで公開された高画質空手動画をキュレーションし、鏡映し・スロー再生・字幕付きで配信する学習プラットフォームです。

## 特徴

### 🥋 CC-BYライセンス動画
- Creative Commons Attribution (CC-BY) ライセンスの動画のみを使用
- 適切な帰属表示と共に学習目的で利用可能
- 高画質な空手の型・組手・基本稽古動画を提供

### 🪞 鏡映し表示
- 動画を左右反転して表示
- まるで鏡を見ているような感覚で学習
- 右利き/左利きに関係なく技を習得可能

### 🐢 可変速再生
- 0.25倍〜2倍速での再生が可能
- 複雑な技の動作をスローモーションで確認
- 自分のペースに合わせた学習

### 📝 字幕機能
- 技の名前や注意点を字幕として表示
- カスタマイズ可能なテキスト
- 理解を深めるための補助情報

### 🎬 動画編集・ダウンロード
- 鏡映し・速度調整・字幕を適用した動画をダウンロード
- FFmpegを使用した高品質な動画処理
- 個人学習用にカスタマイズされた動画の保存

## 技術スタック

## 使用方法

### 基本的な使い方

1. **動画の閲覧**
   - ホームページから「動画を見る」をクリック
   - カテゴリ別（型・組手・基本）でフィルタリング可能

2. **動画の再生**
   - 動画サムネイルをクリックして再生
   - 編集ツールで各種効果を適用

3. **編集機能の使用**
   - 鏡映しボタンで左右反転
   - スライダーで再生速度を調整
   - 字幕テキストを入力して表示

4. **動画のダウンロード**
   - 編集内容を適用した動画をダウンロード
   - 個人学習用として保存可能

### API エンドポイント

```
GET  /api/videos              # 動画一覧の取得
GET  /api/videos/:id          # 特定動画の詳細
POST /api/videos/:id/edit     # 動画編集の実行
GET  /api/download/:filename  # 編集済み動画のダウンロード
POST /api/upload              # 新規動画のアップロード
GET  /api/licenses            # ライセンス情報
GET  /api/stats               # 統計情報
```

## ライセンスについて

### 使用している動画のライセンス
本プラットフォームで使用している動画は、すべて **Creative Commons Attribution (CC BY 4.0)** ライセンスの下で公開されています。

#### CC BY 4.0 の要件
- ✅ 作品の自由な共有・改変
- ✅ 商用利用可能
- ✅ 適切な帰属表示が必要

#### 帰属表示の例
```
作品名: "空手基本型 - 平安初段"
作者: Traditional Karate Foundation
ライセンス: CC BY 4.0
オリジナル: https://example.com/traditional-karate
```

### プラットフォーム自体のライセンス
本ソフトウェアは MIT ライセンスの下で公開されています。
